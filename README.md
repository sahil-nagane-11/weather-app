# 🌦️ Weather App

A responsive weather application built using HTML, CSS, and JavaScript that fetches real-time weather information using the WeatherAPI.

## Features

- Search weather by city
- Current temperature
- Feels like temperature
- Humidity
- Wind speed
- Wind direction
- Pressure
- Visibility
- Sunrise & Sunset
- Weather icons
- Error handling
- Responsive design

## Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
- WeatherAPI

## Getting Started

1. Clone the repository.
2. Get your own API key from WeatherAPI.
3. Replace:

```javascript
const apiKey = "YOUR_API_KEY";
```

with your own API key.

4. Open `index.html` in your browser.
